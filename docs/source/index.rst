alpro
--------------------------------------

alpro (Axion-Like PROpagation) is a python package that allows calculation of survival probability curves due to photon-ALP inter-conversion.

The documentation hosted here will help you install the code, run some basic tests and show you how to calculate some photon-ALP survival probabilities in astrophysical settings.
